# پایتون

پایتون یک زبان برنامه‌نویسی سطح بالا، مفسرری و چندمنظوره است. فلسفه طراحی آن بر خوانایی کد با استفاده از تورفتگی‌های مهم تأکید دارد. پایتون داینامیک تایپ است و از بازیافت حافظه بهره می‌برد.
