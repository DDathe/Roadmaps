# PyPI

مخزن PyPI، که معمولاً "پای‌پی‌آی" تلفظ می‌شود، یک مخزن حاوی چند صد هزار بسته است. این بسته‌ها از پیاده‌سازی‌های ساده مانند "Hello, World" تا کتابخانه‌های پیشرفته یادگیری عمیق را شامل می‌شوند.
