# SQLite

SQLite یک سیستم مدیریت پایگاه داده رابطه‌ای است که در برنامه نهایی جاسازی شده است. این پایگاه داده SQL به صورت خودکفا، بدون سرور، بدون نیاز به پیکربندی و تراکنشی عمل می‌کند.
