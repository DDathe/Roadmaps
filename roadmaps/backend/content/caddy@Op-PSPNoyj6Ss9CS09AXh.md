# Caddy

وب سرور Caddy یک وب سرور قابل توسعه، چندپلتفرمی و متن‌باز است که به زبان Go نوشته شده است. این سرور ویژگی‌های بسیار خوبی مانند SSL/HTTPs خودکار و یک فایل پیکربندی بسیار ساده دارد.
