# What is HTTP?

HTTP یک پروتکل ارتباطی مبتنی بر `TCP/IP` است که نحوه ارتباط بین کلاینت و سرور را استاندارد می‌کند. این پروتکل تعریف می‌کند که محتوا چگونه درخواست و در اینترنت منتقل می‌شود.
