# Database Indexes

**Index** یک ساختار داده است که روی یک جدول موجود ساخته و اختصاص داده می‌شود و به طور اساسی به جستجوی جدول شما می‌پردازد و سعی می‌کند آن را تجزیه و تحلیل کرده و خلاصه کند تا بتواند میانبرهایی ایجاد کند.
